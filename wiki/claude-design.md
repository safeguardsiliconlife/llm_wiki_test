# Claude Design

Anthropic product launched ~April 2026. Generates prototypes, presentations, and reports in a user's corporate visual style by importing design systems from existing assets in one click. Competes directly with [[Figma]]'s non-designer userbase.

## Key Capabilities
- Ingest existing corporate design system assets automatically
- Generate on-brand prototypes, pitch decks, and reports
- Powered by Opus 4.7 with improved vision resolution (2576px / 3.75MP, up from 1568px / 1.15MP)

## Competitive Significance
- Anthropic can run Opus 4.7 at near-zero marginal cost; API customers like Figma pay full price for Sonnet 4.5
- Canva provided a launch testimonial; Figma notably did not
- Built by a small team (~10 people or fewer according to [[Martin Alderson]])
